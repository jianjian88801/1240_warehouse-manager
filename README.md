# 仓库管理系统

#### 介绍
基于JavaWeb实现的仓库管理系统，包含十三个功能模块，满足基本的仓库信息管理功能需求，稍微优化之后可以商用...


#### 软件架构
后端：spring + springmvc + mybatis  
前端：jsp + bootstrap  
环境：jdk1.8 | mysql | maven

#### 使用说明
1. 创建数据库，执行数据库脚本
2. 修改jdbc数据库连接参数
3. 下载安装maven依赖jar
4. 在tomcat服务器部署项目  
    请求地址： http://localhost:8080/warehouse
    用户名：admin  
    密码：123456
